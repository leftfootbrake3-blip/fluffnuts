# Fluffnuts Pro

**Eucalyptus-flavoured Spec-Driven Development prompt engine.**

Fluffnuts Pro is an elite prompt-engineering and surface solution expansion tool. It helps you build structured, spec-driven prompts that produce better results from any AI — Claude, GPT, Gemini, Grok, DeepSeek, and more.

Instead of throwing half-baked requests at an AI and hoping for the best, Fluffnuts Pro guides you through a proper Spec-Driven Development workflow: define your objective, target, context, requirements, constraints, and acceptance criteria — then generate a prompt that actually tells the AI what you need.

## How It Works

**153 FluffyPrompts** across 6 categories — software, content, research, AI, creative, and general — give you a launchpad for any task. Click a chip, fill in the SDD form, and Fluffnuts builds a structured prompt ready to send to your AI of choice.

**Spec-Driven Development (SDD)** turns vague ideas into formal specifications. The SDD panel walks you through each field so nothing gets lost between your brain and the AI's output. The result is a prompt that covers what you need, why you need it, and how you'll know it's done right.

**Multi-AI support.** Configure API endpoints for Claude, OpenAI, Gemini, Grok, DeepSeek, Mistral, and Ollama. Send prompts directly, or copy them to paste wherever you work.

## Features

- 153 built-in prompt chips organised by category
- Spec-Driven Development panel for structured prompt construction
- Custom chip creation — build and save your own prompt templates
- Prompt chaining — combine multiple chips into workflows
- Multi-AI API integration with configurable endpoints
- Prompt history and session management
- Fully offline (except API calls to your chosen AI)
- No telemetry, no subscriptions, no accounts
- Donationware — free to use, pay what you want

## Getting Started

### Run from source (Python 3.11+)

```
pip install flask pystray keyboard pillow
python FluffnutsPro.py
```

### Build the Windows exe

```
pip install cx_Freeze
python setup.py build
```

The exe and all assets land in `dist/`. Run `FluffnutsPro.exe` from there.

## What Fluffnuts Pro Is Not

Fluffnuts Pro is not a code generator or a vibe-coding toy. It's a prompt-engineering tool that helps you communicate precisely with AI systems. The better your spec, the better the output — that's the whole idea.
